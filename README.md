# 경력기술서 템플릿

브라우저에서 바로 편집하고 PDF로 저장할 수 있는 한국어 경력기술서 템플릿입니다.
별도 설치 없이 `index.html` 파일 하나만 열면 바로 사용할 수 있습니다.

---

## 주요 기능

### 1. 편집 모드
화면 우측 하단의 **✏ 편집 모드** 버튼을 클릭하면 편집 모드가 활성화됩니다.
편집 모드에서는 모든 텍스트를 클릭하여 직접 수정할 수 있으며, 상단에 편집 툴바가 표시됩니다.
편집을 마치면 **✓ 편집 완료** 버튼을 클릭하여 편집 모드를 종료합니다.

---

### 2. 인라인 텍스트 편집
편집 모드에서 다음 항목들을 클릭하여 직접 수정할 수 있습니다.

- **헤더 영역**: 이름, 영문이름/직군, 핵심 역할 키워드, 재직 상태, 소속 회사, 포지셔닝 문구
- **태그바**: 핵심 역량 태그 텍스트
- **경력 섹션**: 회사명, 재직 기간, 업종/직책, 역할명, 기간, 업무 내용(불릿), 성과 칩
- **역량 섹션**: 역량 그룹 제목, 역량 항목
- **업무 스타일 섹션**: 업무 스타일 텍스트

> 각 항목에 마우스를 올리면 **툴팁 힌트**가 표시되어 작성 가이드를 확인할 수 있습니다.

---

### 3. 텍스트 서식 (편집 툴바)
편집 모드 상단 툴바에서 텍스트를 선택한 후 서식을 적용할 수 있습니다.

| 기능 | 단축키 |
|------|--------|
| **굵게 (Bold)** | `Ctrl+B` / `⌘+B` |
| *기울임 (Italic)* | `Ctrl+I` / `⌘+I` |
| 텍스트 색상 변경 | 툴바의 **Color** 버튼 클릭 |

색상 팔레트에서 진한, 회색, 연회색, 네이비, 초록, 딥그린, 딥블루, 앰버, 퍼플 등 9가지 색상을 선택할 수 있습니다.

---

### 4. 항목 추가 / 삭제 (전 영역 지원)

편집 모드에서 이력서의 **모든 항목을 자유롭게 추가하거나 삭제**할 수 있습니다.
삭제 시 화면 하단에 **실행취소** 토스트가 표시되어 실수를 복구할 수 있습니다.

#### Core 태그바
| 동작 | 방법 |
|------|------|
| 태그 추가 | 태그바 끝의 **+** 버튼 클릭 → 기본 파란색 태그 생성 |
| 태그 삭제 | 태그 우측의 **×** 버튼 클릭 |
| 태그 색상 변경 | 태그 **우클릭** → 파랑 / 초록 / 보라 / 주황 선택 |
| 태그 순서 변경 | 태그를 마우스로 드래그 |

#### 연락처 (헤더 우측)
| 동작 | 방법 |
|------|------|
| 항목 추가 | **+ 항목 추가** 버튼 클릭 |
| 항목 삭제 | 항목 우측의 **×** 버튼 클릭 |

#### 경력 섹션
| 동작 | 방법 |
|------|------|
| 경력 블록 추가 | 경력 섹션 하단 **+ 경력 추가** 버튼 클릭 (실행취소 가능) |
| 경력 블록 삭제 | 블록 우측 상단의 **✕ 경력** 버튼 클릭 (실행취소 가능) |
| Phase 추가 | 각 회사 블록 하단 **+ Phase 추가** 버튼 클릭 (실행취소 가능) |
| Phase 삭제 | Phase 우측 상단의 **✕ Phase** 버튼 클릭 (실행취소 가능) |
| 불릿 항목 추가 | 불릿 목록 하단 **+ 항목 추가** 버튼 클릭 |
| 불릿 항목 삭제 | 항목 우측의 **✕** 버튼 클릭 |
| 성과 칩 추가 | 성과 행 우측 **+ 추가** 버튼 클릭 |
| 성과 칩 삭제 | 칩 내부의 **✕** 버튼 클릭 |

#### 보유 역량 & 도구 섹션
| 동작 | 방법 |
|------|------|
| 스킬 항목 추가 | 각 그룹 하단 **+ 항목 추가** 버튼 클릭 |
| 스킬 항목 삭제 | 항목 우측의 **×** 버튼 클릭 |
| 그룹 추가 | 섹션 하단 **+ 그룹 추가** 버튼 클릭 (실행취소 가능) |
| 그룹 삭제 | 그룹 제목 좌측의 **×** 버튼 클릭 (실행취소 가능) |

#### 업무 스타일 섹션
| 동작 | 방법 |
|------|------|
| 문단 추가 | 섹션 하단 **+ 문단 추가** 버튼 클릭 |
| 문단 삭제 | 문단 내부 우측의 **×** 버튼 클릭 (실행취소 가능) |

> 모든 추가/삭제 버튼은 **편집 모드에서만 표시**됩니다. 인쇄 및 저장 시 자동으로 제거됩니다.

---

### 5. 드래그 앤 드롭 순서 변경
편집 모드에서 핸들을 드래그하여 항목 순서를 자유롭게 변경할 수 있습니다.

- **회사 블록 순서 변경**: 회사명 좌측의 드래그 핸들(`⠿`)을 드래그
- **Phase 순서 변경**: Phase 제목 좌측의 드래그 핸들을 드래그
- **태그 순서 변경**: 태그바의 태그를 마우스로 드래그하여 좌우 위치 변경

---

### 6. 저장 기능

우측 하단에 3가지 저장 버튼이 제공됩니다.

#### ↓ HTML 저장
현재 경력기술서를 **편집 UI가 제거된 깨끗한 HTML 파일**로 저장합니다.
저장된 파일은 공유 또는 아카이브 용도로 사용할 수 있습니다.
파일명: `career_YYYY-MM-DD.html`

#### ↓ JSON 저장 (`Ctrl+S` / `⌘+S`)
현재 내용을 **JSON 버전 파일**로 저장합니다.
나중에 불러와 이어서 편집할 수 있습니다.
파일명: `career_YYYY-MM-DD.json`

#### ↑ 버전 불러오기
이전에 저장한 JSON 파일을 불러와 내용을 복원합니다.
미저장 변경사항이 있을 경우 확인 팝업이 표시됩니다.

---

### 7. 자동 백업 (로컬 스토리지)
편집 중 변경사항이 발생하면 **30초마다 자동으로 브라우저 로컬 스토리지에 백업**됩니다.
페이지를 다시 열었을 때 이전 자동 백업이 감지되면 상단에 알림 배너가 표시됩니다.

- **복원**: 자동 백업된 내용으로 복원
- **무시**: 자동 백업을 삭제하고 현재 내용 유지

변경사항이 있는 상태에서 페이지를 닫으려 하면 **경고 메시지**가 표시됩니다.

---

### 8. 인쇄 / PDF 저장
화면 하단 또는 편집 모드 종료 후 **인쇄 / PDF 저장** 버튼을 클릭하면
브라우저 인쇄 대화상자가 열립니다. PDF로 저장하려면 프린터를 **"PDF로 저장"** 으로 선택하세요.
인쇄 시 편집 UI(버튼, 핸들 등)는 자동으로 숨겨집니다.

---

### 9. 반응형 레이아웃
640px 이하 모바일 화면에서는 레이아웃이 자동으로 조정됩니다.

- 헤더의 이름/연락처가 세로로 배치됩니다.
- 역량 그리드가 2열로 변경됩니다.

---

## 사용 방법

1. `index.html` 파일을 브라우저로 엽니다.
2. 우측 하단 **✏ 편집 모드** 버튼을 클릭합니다.
3. 각 항목을 클릭하여 본인의 정보를 입력합니다.
4. 필요한 항목은 **+ 버튼**으로 추가, 불필요한 항목은 **× 버튼**으로 삭제합니다.
5. 편집이 완료되면 **↓ HTML 저장** 또는 **↓ JSON 저장**으로 저장합니다.
6. **인쇄 / PDF 저장** 버튼으로 PDF를 생성합니다.

---

## 파일 구조

```
index.html   — 경력기술서 템플릿 (단일 파일, 외부 의존성 없음)
README.md    — 사용 가이드 (본 문서)
```
